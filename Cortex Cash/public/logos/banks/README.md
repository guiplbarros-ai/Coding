# Logos dos Bancos

Adicione as logos dos bancos nesta pasta com os seguintes nomes:

## Formato
- Formato: PNG ou SVG (preferencialmente PNG transparente)
- Tamanho recomendado: 200x200px ou maior
- Fundo transparente

## Arquivos necessários

1. `nubank.png` - Nubank
2. `inter.png` - Inter
3. `itau.png` - Itaú
4. `bradesco.png` - Bradesco
5. `bb.png` - Banco do Brasil
6. `caixa.png` - Caixa Econômica
7. `santander.png` - Santander
8. `c6.png` - C6 Bank
9. `pagbank.png` - PagBank
10. `mercadopago.png` - Mercado Pago
11. `neon.png` - Neon
12. `btg.png` - BTG Pactual
13. `xp.png` - XP Investimentos
14. `modal.png` - Modal
15. `rico.png` - Rico

## Uso

As logos serão acessadas via `/logos/banks/{nome-do-arquivo}.png`
